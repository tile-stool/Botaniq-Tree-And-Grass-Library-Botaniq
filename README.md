# Botaniq – Tree and Grass Library

## Overview

**Botaniq – Tree and Grass Library** is a comprehensive vegetation asset library for Blender developed by Polygoniq. It provides a large collection of optimized 3D trees, grass, flowers, shrubs, palms, ivy, rocks, and other natural assets for architectural visualization, environment design, landscape creation, animation, and game-ready scenes. Built for seamless integration with Blender, Botaniq helps artists quickly populate realistic outdoor environments while maintaining excellent rendering performance and efficient workflows. The library continues to expand with new assets, seasonal variations, scattering tools, and procedural features.

## About

Botaniq is designed to simplify the creation of realistic outdoor scenes in Blender. Instead of manually modeling vegetation, artists can instantly place professionally optimized trees, grass, flowers, and environmental assets into their projects.

The library is suitable for architectural visualization, environmental design, landscape rendering, product visualization, cinematic animation, and many other 3D workflows.
